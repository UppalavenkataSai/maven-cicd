Welcome to Jenkins CI/CD Project

About This Project

This project demonstrates a complete CI/CD pipeline using Jenkins, Maven, GitHub, and Apache Tomcat.

  • Source code is managed using GitHub
  • Build and dependency management handled by Maven
  • Continuous Integration performed by Jenkins
  • Application deployed automatically to Apache Tomcat

Each time the developer pushes code to GitHub, Jenkins automatically:

  1. Clones the repository
  2. Builds the project using Maven
  3. Runs unit tests
  4. Creates a WAR file
  5. Deploys the application to Tomcat

This setup follows real-time DevOps practices and helps ensure faster, reliable, and automated application delivery.
